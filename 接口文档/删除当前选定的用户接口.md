## 简要描述

- 用于删除当前选定的用户接口

## 请求URL

- ` http://localhost:8080/api/delUserByUsername `

## 请求方式

- Delete

## 参数

| 参数名   | 必选 | 类型   | 说明   |
| :------- | :--- | :----- | ------ |
| username | 是   | string | 用户名 |

## 返回示例 

``` java
{
     "code": 200,
     "msg": "操作成功",
     "data": null
}
```
